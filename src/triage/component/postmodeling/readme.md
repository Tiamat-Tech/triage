## Readme for postmodeling

### Functionality that we support
 - for each model group and time (each model id), show:
    - PR-k curve
    - feature importance
    - bias
    - list
       - top k list
       - list cross-tab
       - list descriptives
    - error analysis
 - for each model group, compare
    - feature importances
    - list
    - performance
    - bias
